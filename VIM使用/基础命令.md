# 普通模式 

* x：删除当前字符
* s：替换，即删除当前字符并进入插入模式
* X：向前删除
* c - {motion}：删除从当前位置到motion指示位置的所有内容，并进入插入模型
* S：\^-c-\$
* C: c-\$
* I：\^i
* zz: 重绘屏幕并令当前行居中
* daw：删除当前光标所在的单词以及开头或结尾的空格
* gU：将字符变为大写的操作操作符
* \<C-a\>将当前光标或光标后的第一个数字加一
* \<C-b\>将当前光标或光标后的第一个数字减一
* f-{char}: 查找字符的下一个出现位置，光标停在字符上
* t-{char}：查找字符的下一个出现位置，光标停在字符的前一个字符上
* F-{char}: f的反向
* T-{char}: t的反向
* R: 进入替换模式
# 插入模式
* \<C-h\>:  删除前一个字符（同退格键）
* \<C-w\>: 删除前一个单词
* \<C-u\>: 删至行首
* \<C-o\>: 进入普通-插入模式
* \<C-r\>{register}: 粘贴寄存器register中的内容
* \<C-r\>=：调出表达式寄存器
