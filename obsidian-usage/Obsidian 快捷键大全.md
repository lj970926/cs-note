# Obsidian 快捷键大全

> [!tip] 自定义快捷键
> 所有快捷键均可在 **设置 → 快捷键 (Settings → Hotkeys)** 中自定义。输入框中点击 `×` 可移除绑定，按新的组合键可重新绑定。

---

## 1. 通用

| 快捷键 (macOS) | 快捷键 (Win/Linux) | 功能 |
|:---:|:---:|---|
| `⌘ N` | `Ctrl+N` | 新建笔记 |
| `⌘ P` | `Ctrl+P` | 命令面板 (Command Palette) |
| `⌘ O` | `Ctrl+O` | 快速切换器 (Quick Switcher) |
| `⌘ S` | `Ctrl+S` | 保存当前文件 |
| `⌘ ,` | `Ctrl+,` | 打开设置 |
| `⌘ Q` | `Ctrl+Q` | 退出 Obsidian |
| `⌘ W` | `Ctrl+W` | 关闭当前标签页 |
| `⌘ Shift N` | `Ctrl+Shift+N` | 打开新的 Obsidian 窗口 |
| `⌘ Shift A` | `Ctrl+Shift+A` | 在文件管理器中显示当前文件 |

### 搜索操作符速查

| 操作符 | 示例 | 说明 |
|---|---|---|
| `path:` | `path:folder/` | 搜索指定路径 |
| `file:` | `file:README` | 搜索文件名 |
| `tag:` | `tag:#todo` | 搜索标签 |
| `section:` | `section:标题` | 搜索标题下的内容 |
| `content:` | `content:关键词` | 搜索正文内容 |
| `block:` | `block:(某段话)` | 搜索块引用 |
| `line:` | `line:关键词` | 搜索行内容 |
| `task:` | `task:关键词` | 搜索任务列表 |
| `task-todo:` | `task-todo:关键词` | 搜索未完成任务 |

## 12. Vim 模式快捷键

> [!tip] 启用 Vim 模式
> 在 **设置 → 编辑器 → Vim 键绑定** 中开启。开启后可使用标准 Vim 命令。

### 常用 Vim 命令速查

| 命令 | 模式 | 说明 |
|---|---|---|
| `i` | Normal → Insert | 进入插入模式 |
| `Esc` | Insert → Normal | 返回普通模式 |
| `h j k l` | Normal | 左 下 上 右 移动 |
| `w` / `b` | Normal | 跳到下一个 / 上一个单词 |
| `0` / `$` | Normal | 行首 / 行尾 |
| `gg` / `G` | Normal | 文件开头 / 文件末尾 |
| `dd` | Normal | 删除当前行 |
| `yy` | Normal | 复制当前行 |
| `p` | Normal | 粘贴 |
| `u` | Normal | 撤销 |
| `/关键词` | Normal | 搜索 |
| `:w` | Normal | 保存 |
| `:q` | Normal | 关闭 |
| `Ctrl-c` | Normal | Obsidian 中的 Esc 替代 |

## 15. 自定义快捷键技巧

### 查找并修改快捷键

1. 打开 **设置 → 快捷键 (Settings → Hotkeys)**
2. 在搜索框中输入命令名称
3. 点击 `+` 添加新快捷键，或点击已有快捷键旁的 `×` 移除

### 推荐自定义方案

| 命令 | 推荐快捷键 | 说明 |
|---|---|---|
| 打开日记 (Open daily note) | `⌘ Shift D` | 快速记录 |
| 插入模板 (Insert template) | `⌘ Shift I` | 模板化写作 |
| 切换 Vim 模式 | 通过设置开启 | 提高编辑效率 |
| 打开 Kanban 看板 | `⌘ Shift K` | 项目管理 |
| 打开关系图谱 | `⌘ Shift G` | 查看笔记关联 |
