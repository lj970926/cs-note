> 记录 NuPhy Halo65 的灯光/背光调节快捷键，方便不想翻说明书的自己随时查看。

## 灯光分区

Halo65 的灯光可以分为 **3 个独立区域**：

1. **按键背光**（Backlight）：每个按键下方的 RGB 灯。
2. **环形灯**（Halolight）：键盘四周的 RGB 氛围灯带。
3. **RGB 灯条**（RGB Light Bar）：键盘顶部的状态灯条，也可显示电量、连接模式等。

三个区域的灯效、颜色、亮度、速度都可以独立调节。

---

## 按键背光调节

| 快捷键 | 功能 |
| :--- | :--- |
| `Fn + ← / →` | 切换背光灯效 |
| `Fn + [ / ]` | 切换背光颜色 |
| `Fn + ↑ / ↓` | 增加 / 降低背光亮度 |
| `Fn + < / >` | 加快 / 减慢背光动画速度 |

> `Fn` 键位于空格键右侧。

---

## 环形灯（Halolight）调节

环形灯的快捷键需要先按 `Fn + H`，再搭配方向键或符号键：

| 快捷键 | 功能 |
| :--- | :--- |
| `Fn + H + ← / →` | 切换环形灯灯效 |
| `Fn + H + [ / ]` | 切换环形灯颜色 |
| `Fn + H + ↑ / ↓` | 增加 / 降低环形灯亮度 |
| `Fn + H + < / >` | 加快 / 减慢环形灯动画速度 |

---

## RGB 灯条调节

RGB 灯条的快捷键需要先按 `Fn + /`（问号键），再搭配方向键或符号键：

| 快捷键 | 功能 |
| :--- | :--- |
| `Fn + / + ← / →` | 切换灯条灯效 |
| `Fn + / + [ / ]` | 切换灯条颜色 |
| `Fn + / + ↑ / ↓` | 增加 / 降低灯条亮度 |
| `Fn + / + < / >` | 加快 / 减慢灯条动画速度 |

---

## 电量指示灯

顶部 RGB 灯条也可以用来显示电量：

| 快捷键 | 功能 |
| :--- | :--- |
| `Fn + ]` | 灯条闪烁显示当前电量颜色；再次操作可开启电量指示灯常亮 |
| `Fn + }`（即 `Fn + Shift + ]`） | 关闭电量指示灯常亮 |

> `]` 和 `}` 在同一个物理按键上。官方手册把这三种操作都标在了该键上，实际触发可能因固件版本略有差异，如果按一次没反应可以多试一次或配合 `Shift`。

电量颜色含义：

| 颜色 | 电量 |
| :--- | :--- |
| 绿色 | > 80% |
| 蓝色 | 20% ~ 80% |
| 红色 | < 20% |
| 橙色闪烁 | 充电中 |

---

## 睡眠模式

| 快捷键 | 功能 |
| :--- | :--- |
| `Fn + }` | 开启 / 关闭睡眠模式 |

- 开启睡眠模式后，键盘无操作 **6 分钟** 会自动关灯并进入休眠。
- 无线使用时建议开启，可显著延长续航。

---

## 用 VIA 软件进一步自定义

Halo65 支持 QMK / VIA，可以通过 [VIA](https://usevia.app/) 或 NuPhy 官方工具实现更精细的灯光设置：

- 单键改色 / 自定义 RGB 颜色
- 保存多组灯效配置
- 调整每个灯光区域的详细参数

如果使用 Halo65 V2（Neo / 新版），官方推荐用 **NuPhyIO**（网页驱动）进行改键和灯效配置。

## 其他常用快捷键

| 快捷键 | 功能 |
| :--- | :--- |
| `Fn + X + S` | 切换到 Win 模式 |
| `Fn + X + A` | 切换到 Mac 模式 |
| `Fn + Q / W / E / R` | 切换蓝牙 1 / 2 / 3 / 2.4G |
| `Fn + X + B` | 恢复出厂设置 |

---

## 快速记忆口诀

- **背光**：`Fn + 方向键` 调效果/亮度，`Fn + [ ]` 换颜色，`Fn + < >` 调速度。
- **环形灯**：在背光快捷键前加 `H`。
- **灯条**：在背光快捷键前按 `/`。
