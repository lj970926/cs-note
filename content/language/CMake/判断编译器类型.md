使用 `CMAKE_CXX_COMPILER_ID` 变量来判断编译器类型，`project()` 调用之后即可使用。

常见编译器 ID：
- `"Clang"` — 非 Apple 的 Clang
- `"AppleClang"` — Apple Clang
- `"GNU"` — GCC
- `"MSVC"` — Microsoft Visual C++

**判断是否为 Clang（包括 AppleClang）：**

```cmake
if(CMAKE_CXX_COMPILER_ID MATCHES "Clang")
    # 匹配 Clang 和 AppleClang
endif()
```

**单独判断 Apple Clang：**

```cmake
if(CMAKE_CXX_COMPILER_ID STREQUAL "AppleClang")
    message("Apple Clang")
elseif(CMAKE_CXX_COMPILER_ID STREQUAL "Clang")
    message("其他 Clang")
endif()
```

**结合版本号判断：**

```cmake
if(CMAKE_CXX_COMPILER_ID STREQUAL "Clang" AND CMAKE_CXX_COMPILER_VERSION VERSION_GREATER_EQUAL "14.0")
    message("Clang 14+")
endif()
```

**判断是否为 GCC/G++：**

```cmake
if(CMAKE_CXX_COMPILER_ID STREQUAL "GNU")
    message("这是 GCC/G++")
endif()
```

**GCC 结合版本号判断：**

```cmake
if(CMAKE_CXX_COMPILER_ID STREQUAL "GNU" AND CMAKE_CXX_COMPILER_VERSION VERSION_GREATER_EQUAL "12.0")
    message("GCC 12+")
endif()
```

> [!note]
> `MATCHES` 用的是正则匹配，所以 `"Clang"` 能同时匹配 `"Clang"` 和 `"AppleClang"`；`STREQUAL` 是精确匹配，只匹配完全相同的字符串。
